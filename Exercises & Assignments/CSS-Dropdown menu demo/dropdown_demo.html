Dropdown menu Steps:

  1. Create your navigation links and dropdown 
    <div class="navigation">
	<a class="nav-link" href="#">
		Home
	</a>
	<div class="dropdown">
		<a class="dropdown-top-link" href="#">
			Pages
		</a>
		<div class="dropdown-contents">
			<a class="dropdown-link" href="#">
				Page 1
			</a>
			<a class="dropdown-link" href="#">
				Page 2
			</a>
			<a class="dropdown-link" href="#">
				Page 3
			</a>
		</div>
	</div>
	<a class="nav-link" href="#">
		About Me
	</a>
</div>
  2. In your css, alter the colors/spacing to match your look. 
    .navigation {
	display: flex; /* turns links into columns*/
	width: 960px;
}

.nav-link {
	width: 100px;
	border: 1px solid black;
	height: 50px;
	line-height: 50px;
	background-color: #eee;
	padding: 0 20px;
	color: #000;
	text-decoration: none;
	text-align: center;
}
.nav-link:hover {
	background-color: yellow;
}

.dropdown {
	position: relative;
	background-color: #eee;
}
.dropdown:hover {
	background-color: #ddd;
}
.dropdown-top-link {
	width: 100px;
	display: block;
	height: 50px;
	line-height: 50px;
	padding: 0 20px;
	color: #000;
	text-decoration: none;
	text-align: center;
}


.dropdown-contents {
	display: none; /* hide dropdown contents until hover */
	position: absolute;
	z-index: 2;
	background-color: #ddd;
}
.dropdown:hover .dropdown-contents {
	display: block; /* show contents on hover */
}
.dropdown-link {
	display: block;
	text-decoration: none;
	padding: 10px 20px;
	color: #000;
	width: 200px;
}
.dropdown-link:hover {
	background-color: yellow;
}
