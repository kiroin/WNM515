Slimbox2 demo

Slimbox2 Steps:

  1. Download and link 2 .js files, 1 .css file. (jquery js, slimbox js, lightbox css)
    • Slimbox2
    • Jquery
  2. Add a link to the image you want to show. (can use thumbnails as link body)
  3. Activate link. Add rel="lightbox" to the link code.
    <a href="j1.jpg" rel="lightbox"> <img src="thumbs/j1-sm.jpg"> </a>

Options:

  • Add a title to the link for a caption
    <a href="j1.jpg" rel="lightbox" title="caption text"> link text </a>
  • Group a set of images by adding a -groupname after the rel=lightbox.
    <a href="j1.jpg" rel="lightbox-japan" title="caption text"> link text </a>
    <a href="j1.jpg" rel="lightbox-hawaii" title="caption text"> link text </a>
