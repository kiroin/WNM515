We'll be learning about classes and id's by styling a list of shows!

Instructions:

  1. Select the element body:
    • set it's margin to 0
    • set it's font-family to arial, sans-serif
  2. Select the class media:
    • set it's padding to 10px
    • set it's margin to 50px
  3. Select the class movies:
    • set it's background-color to be orange #E37329
  4. Select the class tv:
    • set it's background-color to be blue #3189B5
  5. Select the class media-title:
    • set the font-size to 50px
    • set the font-weight to normal
    • set the margin to 0
  6. Select the class show-thumbnail:
    • set it's background-color to semi-transparent black rgba(0, 0, 0, 0.3)
    • set the padding: 20px;
    • set the margin: 10px;
    • set the width: 600px;
  7. Select the id mov-6095136 (hot fuzz):
    • set it's background to white #ffffff
    • set the color to black #000000
    • set the text-align: center
    • set the font-weight: bold
