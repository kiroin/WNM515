Slideshow demo

Example using Slides JS

Slides JS Steps:

  1. Download the zip file from Slides JS.
  2. Link 2 .js files
    • Jquery
      <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.5.1/jquery.min.js"></script>
    • Slides.js
      <script type="text/javascript" src="js/slides.min.jquery.js"></script>
  3. Add the HTML content.
    • 3 divs for positioning the content: id=slideshow, id=slides, class=slides_container.
    • The links each become a slide.
    • The 2 links outside the slides_container div become the previous and next buttons
    • The image outside the slides div is the frame
    <div id="slideshow">
	<div id="slides">
		<div class="slides_container">
			<a href=""><img src="http://slidesjs.com/examples/standard/img/slide-1.jpg" alt="Slide 1"></a>
			<a href=""><img src="http://slidesjs.com/examples/standard/img/slide-2.jpg" alt="Slide 2"></a>
			<a href=""><img src="http://slidesjs.com/examples/standard/img/slide-3.jpg" alt="Slide 3"></a>
		</div>
		<a href="#" class="prev"><img src="images/arrow-prev.png" width="24" height="43" alt="Arrow Prev"></a>
		<a href="#" class="next"><img src="images/arrow-next.png" width="24" height="43" alt="Arrow Next"></a>
	</div>
	<img src="images/example-frame.png" width="739" height="341" alt="Example Frame" id="frame">
</div>
  4. Activate the slideshow javascript. Add the following script to your page in the head seaction.
    <script>
	$(document).ready(function(){
		$('#slides').slides({
			preload: true,
			preloadImage: 'images/loading.gif',
			play: 5000,
			pause: 2500,
			hoverPause: true
		});
	});
</script>
  5. Add CSS to style the slideshow
    <style>
