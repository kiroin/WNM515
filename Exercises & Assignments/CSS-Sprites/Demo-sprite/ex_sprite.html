Sprite example


Here is the sprite image in its entirety. Notice how all four states are part of one single image.

By changing the background-position I can selectively mask my image. Rollover the following button to see the code in action.

   

a.btn{
    width: 100px;
    height: 25px;
    display: inline-block;
    background: #2174ef url("images/ex_sprite.gif") 0px 0px;
}
a.btn:hover{ background-position: -200px 0px;}
a.btn:active{ background-position: -300px 0px;}
