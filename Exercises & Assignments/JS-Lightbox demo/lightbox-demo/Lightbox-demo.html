Lightbox demo

Add focus to an image on demand by using lightbox effect. On click, present a larger image above a darkened out page. Uses Lightbox 2 (2.6) and jQuery (1.10.2).

Slimbox2 Steps:

  1. Add a links to the larger images 
    <a href="images/h1.jpg" data-lightbox="hawaii" title="hawaii image">
	<img src="images/thumbs/h1-sm.jpg">
</a>
<a href="images/h2.jpg" data-lightbox="hawaii" title="hawaii image again">
	<img src="images/thumbs/h2-sm.jpg">
</a>
<a href="images/h3.jpg" data-lightbox="hawaii" title="hawaii image again">
	<img src="images/thumbs/h3-sm.jpg">
</a>
<a href="images/h2.jpg" data-lightbox="hawaii" title="hawaii image again">
	<img src="images/thumbs/h4-sm.jpg">
</a>
  2. Link jQuery and Lightbox javascript files. (Add to the head section) 
    <script src="js/jquery-1.10.2.min.js" type="text/javascript"></script>
<script src="js/lightbox/lightbox-2.6.min.js" type="text/javascript"></script>
  3. Link the CSS (You can customize the look of the lightbox within) 
    <link href="js/lightbox/css/lightbox.css" rel="stylesheet" type="text/css" />

Details:

  • Add data-lightbox="" to activate the lightbox. Group a set of photos by using the same data-lightbox name. 
    <a href="j1.jpg" data-lightbox="japan" title="image caption"> link text </a>
<a href="j2.jpg"  data-lightbox="japan" title="image caption"> link text </a>
  • Add a caption by adding title="caption text" 
    <a href="j1.jpg" rel="lightbox" title="caption text"> link text </a>
